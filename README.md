Python Weather App

Overview
This project is a command line based weather application developed using Python. The application allows users to enter a city name and get real time weather details such as temperature weather condition humidity and wind speed. It uses a public weather API to fetch live data and displays it in a readable format.

Purpose of the Project
The main purpose of this project is to help beginners understand how to work with APIs in Python handle user input and display formatted output. It is suitable for academic submissions mini projects and personal practice.

Technologies Used
Python programming language
Requests library for API calls
OpenWeatherMap API for weather data

Project Features
Fetches real time weather information for any valid city
Displays temperature in Celsius
Shows weather condition humidity and wind speed
Handles incorrect or invalid city names gracefully
Simple and easy to understand code structure

Project Structure
The project consists of a single Python file that contains all the logic for fetching and displaying weather data. A README file is included to explain the project usage and setup.

Requirements
Python version 3 or above
Internet connection
Requests library installed

How to Run the Project
First install Python on your system if it is not already installed
Install the requests library using pip
Get a free API key from the OpenWeatherMap website
Open the Python file and replace the placeholder API key with your own key
Run the program using the command python weather_app.py
Enter the city name when prompted to see the weather details

Output Details
The application displays the city name temperature weather description humidity percentage and wind speed. If the city name entered is invalid the program shows an error message asking the user to try again.

Limitations
This application works only with an active internet connection
It provides current weather data only and not future forecasts

Future Enhancements
Add a graphical user interface
Include weather forecast for multiple days
Allow searching weather using zip code or coordinates
Convert the application into a web based project

Conclusion
This Python Weather App is a simple yet effective project for beginners who want to learn API integration and basic application development in Python. It can be easily extended with more features and improvements.
